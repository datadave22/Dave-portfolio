PROMPT START

This is a MOTION & SCROLL BEHAVIOR REFINEMENT PASS.
Do NOT redesign layout, color, or typography.
Do NOT remove existing content.

Your task is to implement a Cosmos-style scroll experience using geometric forms instead of images.

────────────────────────────
1. Scroll Model (CRITICAL)
────────────────────────────

Implement SECTION-BASED SCROLL CHOREOGRAPHY.

Rules:
- The page scrolls normally, but each major section behaves as a “scene”
- Scroll transitions occur at section boundaries
- Each section has:
  - Entry state
  - Active state
  - Exit state

NO continuous parallax.
NO infinite floating motion.

Scroll should feel:
- Weighted
- Intentional
- Editorial
- Architectural

────────────────────────────
2. Scene Transition Behavior
────────────────────────────

On scroll INTO a section:
- Primary geometry enters from Y-axis (±40–80px)
- Opacity transitions from 0 → 1
- Duration: 600–900ms
- Easing: cubic-bezier(0.4, 0.0, 0.2, 1)

On scroll OUT of a section:
- Geometry exits with inverse motion
- Slight scale reduction (1 → 0.98)
- Opacity fades gently (1 → 0)

Only ONE primary animated element per section.
Secondary elements remain mostly static.

────────────────────────────
3. Geometry as Content Drivers
────────────────────────────

Replace image-based motion patterns with:

- Lines
- Rectangular planes
- Modular blocks
- Circles used sparingly

Geometry must:
- Define spatial rhythm
- Anchor content
- Move as structural elements, not decoration

Examples:
- A horizontal line that extends on section entry
- A rectangular plane that slides in behind text
- A circle that reveals content via mask or clip-path

NO ornamental animation.
NO symbolism.
NO mystical shapes.

────────────────────────────
4. Motion Tokens (MANDATORY)
────────────────────────────

Define and reuse these tokens globally:

EASING:
- Standard: cubic-bezier(0.4, 0.0, 0.2, 1)
- Exit: cubic-bezier(0.4, 0.0, 1, 1)

DURATION:
- Fast: 300ms (micro-interactions only)
- Standard: 600ms
- Slow: 900ms (hero / section transitions)

DISTANCE:
- Small: 24px
- Medium: 48px
- Large: 72px

Scale:
- Entry: 0.98 → 1
- Exit: 1 → 0.98

NO other easing curves permitted.

────────────────────────────
5. Hero Section (Special Handling)
────────────────────────────

Hero must feel like an opening editorial spread.

Behavior:
- Geometry loads first
- Text fades in second
- Scroll down causes:
  - Geometry to move upward slowly
  - Content to remain anchored
  - No collapse or jump

Hero motion must suggest depth without 3D.

────────────────────────────
6. Hover Interactions
────────────────────────────

Hover behavior is minimal:

- Slight opacity shift OR
- 1–2px translate OR
- Line weight emphasis

Duration: 150–200ms
Easing: standard token only

NO scaling above 1.02
NO glow
NO bounce

────────────────────────────
7. Accessibility & Performance
────────────────────────────

- Respect prefers-reduced-motion
- Motion must be transform + opacity only
- No layout thrashing
- No scroll jank

────────────────────────────
8. Anti-Patterns (STRICTLY FORBIDDEN)
────────────────────────────

- Continuous parallax backgrounds
- Auto-looping animations
- Randomized motion
- Multiple competing animations per section
- Image placeholders posing as geometry
- Overlapping motion timelines
- “Cool” effects that do not serve hierarchy

────────────────────────────
9. Quality Check
────────────────────────────

Before finalizing:
- Scroll through the site slowly
- Ensure each section feels distinct
- Motion should feel editorial, not playful
- If motion feels noticeable, reduce it

Motion should be felt, not seen.

PROMPT END
